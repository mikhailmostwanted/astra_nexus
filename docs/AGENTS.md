# Agents

## CoordinatorAgent

Разбивает задачу на маршрут работы. Не пишет финальный результат, а задаёт порядок и
критерии готовности.

## ResearcherAgent

Собирает факты, ограничения и исходный материал. В MVP не ходит в интернет и не
вызывает платные API.

## WriterAgent

Готовит черновик результата на основе плана и исследования.

## CriticAgent

Проверяет черновик на явные пробелы, риски и несоответствия задаче. Не спорит ради
спора и не расширяет задачу без причины.

## FinalizerAgent

Собирает итоговый ответ или файл для пользователя.

## Ограничение бесконечного спора

- Порядок агентов фиксирован для MVP.
- Каждый агент получает один ход.
- Critic даёт только список значимых замечаний.
- Finalizer принимает итоговое решение и закрывает run.
- Оркестратор, а не агенты, управляет состоянием задачи.
