# Roadmap

## Итерация 1

Архитектурный скелет: backend, SQLite, dummy agents, workspace, тесты и документация.

## Итерация 2

Рабочий Telegram task flow: polling, команды, статусы, вывод сообщений агентов.

## Итерация 3

NoDriver + ChatGPT Web provider с безопасным локальным browser profile.

## Итерация 4

Файлы задач: входные документы, Markdown, DOCX, PDF и отправка артефактов в Telegram.

## Итерация 5

Mini app/dashboard для обзора задач, runs, агентов, артефактов и настроек.
